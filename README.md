# CS 学习笔记

CS 核心课程学习笔记与代码练习。

## 科目

| 目录 | 教材 | 语言 |
|------|------|------|
| `sedgewick/` | 算法（第四版） | Java |
| `csapp/` | CS:APP（第三版） | C |
| `ostep/` | OSTEP | C |
| `topdown/` | 计算机网络：自顶向下方法 | Python/C |

## 目录结构

每个科目独立目录，包含 `notes/`（学习笔记）和 `codes/`（代码实现）。

`site/` 是基于 [Fumadocs](https://fumadocs.vercel.app/) 的统一文档站。

## 本地开发

```bash
cd site && bun install && bun run dev
```
